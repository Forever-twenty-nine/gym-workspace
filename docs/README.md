## 🛠️ Tecnologías

- **Frontend**: Angular 20, Ionic 8, TypeScript 5.8
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Mobile**: Capacitor, Android SDK
- **Testing**: Jasmine, Karma

- **gym-admin**: Panel administrativo web
- **gym-app**: App móvil Ionic
