# Funcionalidades para Entrenadores

La aplicación móvil proporciona herramientas potentes para que los entrenadores gestionen a sus clientes y programas de entrenamiento de manera eficiente.

## Características Principales

### 📊 Dashboard de Gestión
- Vista general del rendimiento de los alumnos.
- Alertas sobre alumnos inactivos o que requieren atención.
- Resumen de la jornada y próximas citas.

### 📅 Agenda y Citas
- Calendario interactivo para programar sesiones personales.
- Gestión de disponibilidad y cancelaciones.
- Recordatorios automáticos para entrenadores y clientes.

### 👥 Gestión de Entrenados
- Listado completo de clientes asignados.
- Perfil detallado de cada alumno (objetivos, lesiones, historial).
- Chat directo o integración de contacto para seguimiento.

### 📝 Constructor de Rutinas
- Herramienta para crear rutinas personalizadas de forma rápida.
- Biblioteca de plantillas para diferentes perfiles (fuerza, hipertrofia, pérdida de peso).
- Asignación masiva o individual de rutinas.

### 📚 Librería de Ejercicios
- Diccionario completo de ejercicios con descripciones y multimedia.
- Posibilidad de añadir variantes personalizadas.
- Clasificación por grupo muscular y equipamiento necesario.
