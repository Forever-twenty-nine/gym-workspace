# Funcionalidades para Gimnasios (Administración)

El módulo institucional permite a los dueños y administradores de gimnasios supervisar la operación global desde su dispositivo móvil.

## Características Principales

### 🏛️ Dashboard Institucional
- Métricas clave de rendimiento (KPIs).
- Control de flujo de usuarios en tiempo real.
- Resumen de suscripciones activas y vencimientos.

### 🗓️ Agenda Global
- Gestión de horarios de clases grupales.
- Reserva de espacios y equipamiento especial.
- Calendario de eventos y cierres programados.

### 👨‍🏫 Gestión de Staff (Entrenadores)
- Control de asistencia y desempeño de los entrenadores.
- Asignación de alumnos a entrenadores específicos.
- Gestión de perfiles profesionales dentro de la plataforma.

### 👥 Administración de Usuarios
- Registro centralizado de todos los miembros del gimnasio.
- Control de acceso y estado de membresías.
- Segmentación de usuarios para comunicaciones institucionales.
